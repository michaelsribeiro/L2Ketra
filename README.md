<h2>How to use 🤔</h2>

```
- Clone this repo:
  $ git clone https://github.com/michaelsribeiro/L2Ketra.git

- Install the project in the htdocs folder:
  $ C:/xampp/htdocs/l2Ketra

- Turn on xampp server

- Open your browser in:
  http://localhost/L2Ketra/ 
```
